Install windows git:
====================
1. Open terminal.
2. run command `winget install --id Git.Git -e --source winget`
3. Check version by running command `git --version`

4. Set global user and email by running 
`git config --global user.name "FirstName SecondName"`
`git config --global user.email "Your email"`
5. Checking global config list
`git config list --global`
